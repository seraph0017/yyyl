## 3. 实现阶段注意

| 编号 | 要求 |
|------|------|
| AD-01 | 导出下载必须使用 blob 鉴权接口，不能使用静态 URL |
| AD-02 | 全额退款不取消订单必须隐藏给非 super_admin |
| AD-03 | `pages/category/index` 是小程序 tabBar，二维码说明里不要展示不可用的 redirectTo 路径 |
| AD-04 | 高级筛选条件较多，移动宽度下应折叠，不要撑破页面 |
| AD-05 | CMS 页面二维码按钮必须以 `current_version_id` 判断发布状态 |
