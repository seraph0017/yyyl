# v1.6 管理后台（admin）开发文档

> **版本**：v1.6
> **日期**：2026-03-31
> **基线**：PRD v1.6-rev1 增量需求
> **目标读者**：Admin 前端开发 Agent

---

## 1. 概述

本次管理后台开发范围为 **CMS 可视化装修编辑器** 和 **企业宣传首页**，核心工作包括：

1. **页面装修列表** — 页面 CRUD 管理
2. **可视化装修编辑器** — 三栏画板（左组件面板 + 中手机画布 + 右属性面板），支持拖拽排序、撤销/重做、自动保存、编辑锁
3. **11 种组件属性配置表单** — 每种 CMS 组件的配置面板
4. **链接选择器** — 7 种链接类型的配置弹窗
5. **素材库** — 图片/视频上传与管理
6. **版本管理** — 版本列表与回滚
7. **企业宣传首页** — 全屏品牌宣传页 + 登录弹窗 + 桌面端 CMS 渲染引擎
8. **路由守卫改造** — 未登录默认跳转宣传页（非登录页）

**不在本端范围内**：
- 后端 API 开发（server 端负责）
- 小程序端 CMS 渲染（uni-app 端负责）

### 路由守卫改造

```typescript
// 修改路由守卫的未登录逻辑
router.beforeEach(async (to, _from, next) => {
  NProgress.start()
  document.title = `${to.meta.title || '管理后台'} - ${brandConfig.name}`

  const token = getToken()

  if (to.meta.public) {
    // 公共页面
    if (token) {
      // 已登录用户访问公共页 → 跳 Dashboard
      if (to.path === '/login' || to.path === '/landing') {
        next('/dashboard')
      } else {
        next()
      }
    } else {
      next()
    }
  } else {
    // 需要登录的页面
    if (!token) {
      // ★ 改造点：未登录跳 /landing（原来跳 /login），保留原始目标路径
      next(`/landing?redirect=${encodeURIComponent(to.fullPath)}`)
    } else {
      // ... 原有的 token 验证和角色检查逻辑不变
      const userStore = useUserStore()
      if (!userStore.isVerified) {
        const isValid = await userStore.verifyAndRefreshUser()
        if (!isValid) {
          next(`/landing?redirect=${encodeURIComponent(to.fullPath)}`)
          return
        }
      }
      const requiredRoles = to.meta.roles as string[] | undefined
      if (requiredRoles) {
        if (userStore.hasRole(requiredRoles)) {
          next()
        } else {
          next('/dashboard')
        }
      } else {
        next()
      }
    }
  }
})
```

### 6.1 页面装修列表 (`views/cms/index.vue`)

- 顶部：筛选栏（页面类型下拉 + 搜索框 + 新建按钮）

**表格列**：

| 列 | 字段 | 说明 |
|----|------|------|
| 页面标识 | page_code | — |
| 页面标题 | title | — |
| 页面类型 | page_type | 标签展示 |
| 状态 | status | active=绿色/inactive=灰色 |
| 发布状态 | current_version_id | 有值=「已发布」绿色标签，无=「未发布」灰色 |
| 草稿更新 | draft_updated_at | 时间格式化 |
| 操作 | — | 编辑（跳编辑器）、设置（编辑基本信息弹窗）、删除（仅custom类型） |

**交互**：
- 「编辑」按钮 → `router.push({ name: 'CmsEditor', params: { id } })`
- 「新建」→ el-dialog 弹窗填写 page_code/page_type/title
- 「删除」→ 确认弹窗，仅 custom 类型可删

### 6.2 可视化装修编辑器 (`views/cms/editor.vue`)

**三栏布局**：

```
┌─────────────────────────────────────────────────────┐
│  工具栏：← 返回 | 页面标题 | 保存草稿 | 预览 | 发布 | 版本 | ↩ ↪  │
├───────────┬───────────────────┬──────────────────────┤
│  组件面板   │     手机画布       │     属性面板          │
│  (240px)   │   (375×667)       │     (320px)          │
│            │                   │                      │
│  基础组件   │  [Banner 轮播图]  │  当前选中组件的        │
│  · Banner  │  [快捷导航]       │  属性配置表单          │
│  · 图片    │  [商品列表]       │                      │
│  · 图文    │  [公告栏]         │  + 通用样式设置        │
│  · 公告    │                   │  （外边距、圆角、背景） │
│  · 导航    │  拖拽可排序        │                      │
│  · 商品    │                   │                      │
│  营销组件   │                   │                      │
│  · 优惠券  │                   │                      │
│  辅助组件   │                   │                      │
│  · 富文本  │                   │                      │
│  · 间距    │                   │                      │
│  · 分割线  │                   │                      │
│  · 视频    │                   │                      │
├───────────┴───────────────────┴──────────────────────┤
│  状态栏：自动保存状态 | 组件数量 x/30 | 编辑锁状态          │
└─────────────────────────────────────────────────────┘
```

**数据流**：
1. 进入编辑器 → 获取编辑锁 → 加载页面详情 → 初始化 store → 根据 `page.page_type === 'landing'` 设置 `previewMode = 'desktop'`（默认 `'mobile'`）
2. 拖拽/编辑 → 更新 store.config → 画布实时刷新
3. 60 秒自动保存草稿 → 调用 API
4. 点击发布 → 确认弹窗 → 调用发布 API（非 `super_admin` 时发布/回滚按钮 `disabled` + tooltip 提示「仅超级管理员可发布页面」，使用 `useUserStore().isSuperAdmin` 判断）
5. 离开编辑器 → 释放编辑锁

## 8. 状态管理 (`stores/cms.ts`)

```typescript
import { defineStore } from 'pinia'
import { ref, computed } from 'vue'
import type { CmsConfig, ComponentItem, CmsPage } from '@/types/cms'
import { v4 as uuidv4 } from 'uuid'

export const useCmsStore = defineStore('cms', () => {
  // ---- State ----
  const page = ref<CmsPage | null>(null)
  const config = ref<CmsConfig>({
    version: 1,
    page_settings: {
      background_color: '#faf6f0',
      title_bar_color: '#2d4a3e',
      title_bar_text_color: '#ffffff',
    },
    components: [],
  })
  const selectedComponentId = ref<string | null>(null)
  const undoStack = ref<string[]>([])     // JSON快照栈
  const redoStack = ref<string[]>([])
  const isDirty = ref(false)              // 是否有未保存修改
  const lastSavedAt = ref<string | null>(null)
  const clipboard = ref<ComponentItem | null>(null)  // 组件剪贴板
  const readonly = ref(false)             // 只读模式标志（编辑锁冲突时启用）

  // ---- Computed ----
  const selectedComponent = computed(() => {
    if (!selectedComponentId.value) return null
    return config.value.components.find(c => c.id === selectedComponentId.value) || null
  })

  const componentCount = computed(() => config.value.components.length)

  // ---- 撤销/重做 ----
  const MAX_UNDO = 50

  function pushSnapshot() {
    const snapshot = JSON.stringify(config.value)
    undoStack.value.push(snapshot)
    if (undoStack.value.length > MAX_UNDO) {
      undoStack.value.shift()
    }
    redoStack.value = []  // 新操作清空重做栈
    isDirty.value = true
  }

  function undo() {
    if (undoStack.value.length === 0) return
    redoStack.value.push(JSON.stringify(config.value))
    const prev = undoStack.value.pop()!
    config.value = JSON.parse(prev)
    isDirty.value = true
  }

  function redo() {
    if (redoStack.value.length === 0) return
    undoStack.value.push(JSON.stringify(config.value))
    const next = redoStack.value.pop()!
    config.value = JSON.parse(next)
    isDirty.value = true
  }

  // ---- 组件操作 ----

  function addComponent(type: string, defaultProps: Record<string, any>, index?: number) {
    // 组件数量上限30检查
    if (config.value.components.length >= 30) {
      ElMessage.warning('组件数量已达上限（30个），请拆分页面')
      return
    }
    pushSnapshot()
    const newComp: ComponentItem = {
      id: uuidv4(),
      type,
      props: { ...defaultProps },
      style: { margin_top: 0, margin_bottom: 10 },
    }
    if (index !== undefined) {
      config.value.components.splice(index, 0, newComp)
    } else {
      config.value.components.push(newComp)
    }
    selectedComponentId.value = newComp.id
  }

  function removeComponent(id: string) {
    pushSnapshot()
    const idx = config.value.components.findIndex(c => c.id === id)
    if (idx > -1) {
      config.value.components.splice(idx, 1)
      if (selectedComponentId.value === id) {
        selectedComponentId.value = null
      }
    }
  }

  function moveComponent(fromIndex: number, toIndex: number) {
    pushSnapshot()
    const [item] = config.value.components.splice(fromIndex, 1)
    config.value.components.splice(toIndex, 0, item)
  }

  function duplicateComponent(id: string) {
    pushSnapshot()
    const idx = config.value.components.findIndex(c => c.id === id)
    if (idx > -1) {
      const copy: ComponentItem = JSON.parse(JSON.stringify(config.value.components[idx]))
      copy.id = uuidv4()
      config.value.components.splice(idx + 1, 0, copy)
      selectedComponentId.value = copy.id
    }
  }

  function copyToClipboard(id: string) {
    const comp = config.value.components.find(c => c.id === id)
    if (comp) {
      clipboard.value = JSON.parse(JSON.stringify(comp))
    }
  }

  function pasteFromClipboard() {
    if (!clipboard.value) return
    if (config.value.components.length >= 30) {
      ElMessage.warning('组件数量已达上限（30个），请拆分页面')
      return
    }
    pushSnapshot()
    const pasted: ComponentItem = JSON.parse(JSON.stringify(clipboard.value))
    pasted.id = uuidv4()
    // 粘贴到选中组件后面，或列表末尾
    const idx = selectedComponentId.value
      ? config.value.components.findIndex(c => c.id === selectedComponentId.value)
      : -1
    if (idx > -1) {
      config.value.components.splice(idx + 1, 0, pasted)
    } else {
      config.value.components.push(pasted)
    }
    selectedComponentId.value = pasted.id
  }

  function updateComponentProps(id: string, props: Record<string, any>) {
    pushSnapshot()
    const comp = config.value.components.find(c => c.id === id)
    if (comp) {
      comp.props = { ...comp.props, ...props }
    }
  }

  function updateComponentStyle(id: string, style: Partial<ComponentItem['style']>) {
    pushSnapshot()
    const comp = config.value.components.find(c => c.id === id)
    if (comp) {
      comp.style = { ...comp.style, ...style }
    }
  }

  // ---- 初始化/重置 ----

  function initFromPage(pageData: CmsPage) {
    page.value = pageData
    if (pageData.draft_config) {
      config.value = JSON.parse(JSON.stringify(pageData.draft_config))
    } else {
      config.value = {
        version: 1,
        page_settings: { background_color: '#faf6f0', title_bar_color: '#2d4a3e', title_bar_text_color: '#ffffff' },
        components: [],
      }
    }
    undoStack.value = []
    redoStack.value = []
    isDirty.value = false
    selectedComponentId.value = null
  }

  function $reset() {
    page.value = null
    config.value = { version: 1, page_settings: { background_color: '#faf6f0', title_bar_color: '#2d4a3e', title_bar_text_color: '#ffffff' }, components: [] }
    selectedComponentId.value = null
    undoStack.value = []
    redoStack.value = []
    isDirty.value = false
    lastSavedAt.value = null
    clipboard.value = null
    readonly.value = false
  }

  return {
    page, config, selectedComponentId, undoStack, redoStack, isDirty, lastSavedAt, clipboard, readonly,
    selectedComponent, componentCount,
    pushSnapshot, undo, redo,
    addComponent, removeComponent, moveComponent, duplicateComponent,
    copyToClipboard, pasteFromClipboard,
    updateComponentProps, updateComponentStyle,
    initFromPage, $reset,
  }
})
```

## 10. 撤销/重做方案

- **策略**：每次修改 config 前，push 当前 config 的 JSON 快照到 undoStack
- **最大步数**：50 步，超出后删除最早的快照
- **重做栈**：每次新操作时清空 redoStack
- **快捷键**：在 editor.vue 中监听键盘事件

```typescript
// editor.vue
function handleKeydown(e: KeyboardEvent) {
  if ((e.ctrlKey || e.metaKey) && e.key === 'z' && !e.shiftKey) {
    e.preventDefault()
    cmsStore.undo()
  }
  if ((e.ctrlKey || e.metaKey) && e.key === 'z' && e.shiftKey) {
    e.preventDefault()
    cmsStore.redo()
  }
  if ((e.ctrlKey || e.metaKey) && e.key === 's') {
    e.preventDefault()
    saveDraft()
  }
  // 复制选中组件到剪贴板
  if ((e.ctrlKey || e.metaKey) && e.key === 'c' && cmsStore.selectedComponentId) {
    e.preventDefault()
    cmsStore.copyToClipboard(cmsStore.selectedComponentId)
  }
  // 粘贴剪贴板中的组件
  if ((e.ctrlKey || e.metaKey) && e.key === 'v') {
    e.preventDefault()
    cmsStore.pasteFromClipboard()
  }
  if (e.key === 'Delete' && cmsStore.selectedComponentId) {
    cmsStore.removeComponent(cmsStore.selectedComponentId)
  }
}

onMounted(() => document.addEventListener('keydown', handleKeydown))
onUnmounted(() => document.removeEventListener('keydown', handleKeydown))
```

## 11. 自动保存与编辑锁

### 自动保存草稿（60秒）

```typescript
// editor.vue
let autoSaveTimer: ReturnType<typeof setInterval> | null = null

async function saveDraft() {
  if (!cmsStore.isDirty || !cmsStore.page) return
  try {
    const res = await saveCmsDraft(cmsStore.page.id, {
      config: cmsStore.config,
      draft_updated_at: cmsStore.page.draft_updated_at,
    })
    // 更新乐观锁时间戳为服务端返回值，避免下次保存误判冲突
    cmsStore.page!.draft_updated_at = res.data.draft_updated_at
    cmsStore.isDirty = false
    cmsStore.lastSavedAt = new Date().toISOString()
    ElMessage.success('草稿已保存')
  } catch (err: any) {
    if (err.response?.status === 409) {
      ElMessageBox.alert('页面已被他人修改，请刷新后重试', '保存冲突')
    }
  }
}

onMounted(() => {
  autoSaveTimer = setInterval(saveDraft, 60000)  // 60秒
})

onUnmounted(() => {
  if (autoSaveTimer) clearInterval(autoSaveTimer)
})
```

### 编辑锁心跳（2分钟）

```typescript
let heartbeatTimer: ReturnType<typeof setInterval> | null = null

// 页面关闭/刷新时释放编辑锁（onUnmounted 在标签页直接关闭时不一定触发）
function handleBeforeUnload() {
  // sendBeacon 在页面关闭时可靠发送
  const url = `/api/v1/admin/cms/pages/${pageId}/lock`
  const token = getToken()
  // 方案1：使用 fetch keepalive（可携带自定义 header）
  fetch(url, {
    method: 'DELETE',
    headers: { 'Authorization': `Bearer ${token}`, 'X-Site-Id': String(siteId) },
    keepalive: true,
  }).catch(() => {})
}

onMounted(async () => {
  // 获取编辑锁
  try {
    await acquireCmsLock(pageId)
  } catch (err: any) {
    if (err.response?.status === 423) {
      const lockInfo = err.response.data.detail?.data
      try {
        // 提供只读查看选项
        await ElMessageBox.confirm(
          `该页面正在被 ${lockInfo?.admin_name || '其他管理员'} 编辑中`,
          '编辑锁冲突',
          { confirmButtonText: '只读查看', cancelButtonText: '返回列表', type: 'warning' }
        )
        // 用户选择只读查看
        cmsStore.readonly = true
        // 继续加载页面数据，但禁用拖拽/编辑/保存/发布
      } catch {
        // 用户选择返回列表
        router.back()
      }
      return
    }
  }

  // 心跳续期
  heartbeatTimer = setInterval(async () => {
    try {
      await acquireCmsLock(pageId)
    } catch { /* 静默失败 */ }
  }, 120000)  // 2分钟

  // 注册 beforeunload 监听器
  window.addEventListener('beforeunload', handleBeforeUnload)
})

onUnmounted(async () => {
  if (heartbeatTimer) clearInterval(heartbeatTimer)
  window.removeEventListener('beforeunload', handleBeforeUnload)
  // 释放编辑锁
  try {
    await releaseCmsLock(pageId)
  } catch { /* 静默 */ }
})
```

## 14. 与其他端的接口约定

### 后端 API 调用约定
- 所有 B端接口前缀 `/admin/cms/`，自动携带 JWT token 和 `X-Site-Id` header
- 宣传页公开接口 `GET /cms/landing` 无需 token，通过 query 参数传 site_id
- 统一响应格式：`{ code: 0, message: "success", data: {...} }`
- 分页响应：`{ data: { list: [...], pagination: { page, page_size, total, total_pages } } }`
- 错误响应：`{ code: "CMS_XXX", message: "错误描述" }`

### 与小程序端共用 JSON 配置格式
- **配置 JSON 格式完全一致**：`CmsConfig` 类型定义为准
- Admin 端生成/编辑配置 JSON → 后端存储 → 小程序端读取并渲染
- 组件 `type` 标识统一使用英文标识：banner、image、image_text、notice、nav、product_list、coupon、rich_text、spacer、divider、video
- 链接配置统一使用 `LinkConfig` 格式
